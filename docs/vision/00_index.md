## Folder meanings

- `canon/` — CHAD-approved wording + binding constraints.
- `doctrine/` — guardrails, red lines, decision heuristics (versioned).
- `research/` — case studies, patterns, long-horizon ideas.
- `working/` — drafts and explorations; not binding.
- `archive/` — immutable historical artifacts; date-stamped.
