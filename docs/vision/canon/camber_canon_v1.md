# Camber Canon v1

Status: ACTIVE
Effective: 2026-02-17
Owner: CHAD

This document is the binding constraint set for Camber strategy and product decisions.

## Canonical mission statement

Camber turns real communications into durable, evidence-backed project memory and actionable intelligence. It drives better decisions across communication, scheduling, relationship, and financial workflows.

## Binding constraints

1) Internal-only framing
- Any strong asymmetry language is internal-only.
- "Slack for contractors" is internal-only.

2) Primary user
- CHAD + ZACK.

3) Receipts
- Every promoted item must include transcript-span receipts (`transcript_span`).

4) Review lane
- Review owner: CHAD.
- Review budget target: 10 items/day.

5) Scoring / analytics
- No external scoring of subs/vendors.
- Internal analytics is allowed but long-horizon.

6) Sharing model
- One underlying truth ledger with permissioned projections by role/relationship.

7) Governance
- Mission/vision positioning requires CHAD approval.
