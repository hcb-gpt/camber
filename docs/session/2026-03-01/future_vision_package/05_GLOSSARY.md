# 05 — Glossary

A few terms used consistently across the vision.

- Evidence (grounded): Something the user can inspect (e.g., transcript excerpt, explicit artifact) that supports a claim.
- Proxy signal: A heuristic (e.g., affinity, prior) that can guide guesses but should not be promoted as truth.
- World model: The evolving representation of projects, entities, and relationships that Camber uses to reason.
- Promotion: Writing a belief into a durable state where it affects future decisions.
- Gate: A rule that blocks promotion/automation unless conditions are met.
- Circuit breaker: A monitoring rule that freezes learning/automation when quality degrades.
- Continuity: The idea that multiple communications belong to the same thread/topic; not automatically the same project.
- Habit surface: A UI surface that becomes part of daily workflow (opened and acted on).
